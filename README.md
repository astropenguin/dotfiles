# dotfiles

Personal dotfiles managed by rcm

## Usage

```shell
$ brew install rcm
$ git clone --recursive https://github.com/astropenguin/dotfiles.git $HOME/.dotfiles
$ env RCRC=$HOME/.dotfiles/rcrc rcup
```
